GitHub

the things I love about you

GitHub, you're always there, far away yet close enough to fork. Your presence looms over me like a... cloud. Me, I'm only human - sometimes I push, sometimes I pull - but you always stay committed. You always back me up - you maintain the best version of myself. And that is why I love you.

  • Create an account
  • Create a remote repository
  • Use 'git remote' command to link local repo to remote
  • Use 'git push <remote_name> <branch_name>' to push local commits to GitHub'
  • Use 'git pull' to pull upstream commits on a branch
